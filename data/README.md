# Data directory

## Purpose
Store all raw and processed inputs needed for manuscript reproduction.

## Required documentation before release
- Provenance of each dataset
- Access restrictions or licensing
- Data schema / column definitions
- Version identifier or snapshot date
- Transformation rules from raw to processed

## Recommended subfolders
- `raw/` for immutable intake files
- `processed/` for analysis-ready derived files
- `external/` for linked or downloaded third-party assets if licensing permits
